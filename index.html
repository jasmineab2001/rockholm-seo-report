Keyword Clustering Logic
Keywords with the same topic + intent are grouped into one page. Ranking for the focus keyword pulls rankings for all the others automatically. Example from the data: ayurveda kerala (260) · ayurveda kerala resort (170) · ayurveda resort kerala (110) — all three target one page at /de/ayurveda-kerala-resort/ with combined reach of ~540/mo. Every keyword in the Excel is assigned to exactly one cluster below.
