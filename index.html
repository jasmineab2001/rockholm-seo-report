📋 Strategy Per Category

🆕 New Pages — What We Build
  • ✓ Title tag (50–58 chars) with primary keyword
  • ✓ Meta description (150–155 chars) with CTA
  • ✓ H1 targeting the main keyword
  • ✓ H2/H3 structure covering all search intent angles
  • ✓ 1,200–2,000 words of original content
  • ✓ 5–8 images with keyword-aware alt text
  • ✓ 5–6 FAQ pairs with schema markup
  • ✓ 2–3 CTAs (Book Now / View Packages / Enquire)
  • ✓ 3–5 internal links to related pages
  • ✓ Submit URL to GSC after publish
🔧 Rework Pages — What We Update
  • ✓ Keyword optimization — add primary + secondary keywords naturally
  • ✓ Content optimization — expand thin content to 1,200–2,000 words
  • ✓ Meta tag rewrite — new title + description
  • ✓ URL optimization — 301 redirect if slug is not keyword-friendly
  • ✓ Internal links — add 3–5 links to/from related pages
  • ✓ FAQ schema — add or improve structured markup
  • ✓ Image alt text — update all alt attributes
✍️ New Blogs — How We Write for AI + Google
  • ✓ Clear direct answer in first 100 words (AI snippet bait)
  • ✓ Keyword-optimized title + short slug + meta description
  • ✓ H2/H3 structure covering all related questions
  • ✓ 1,500–2,500 words of factual, well-structured content
  • ✓ 5–8 FAQs with schema (Google + AI search ready)
  • ✓ 2–3 internal links to commercial pages (Panchakarma, Prices, Book)
  • ✓ Author attribution + Rockholm expert quote
  • ✓ Optimized slug — short, keyword-focused URL
🔄 Rework Blogs — What We Improve
  • ✓ Meta tag optimization — rewrite title + meta with target keyword
  • ✓ Slug optimization — short keyword-focused URL + 301 redirect
  • ✓ Content update — add missing sections, expand thin content
  • ✓ Keyword refresh — add primary + secondary keywords naturally
  • ✓ FAQ schema — add or expand structured FAQ markup
  • ✓ Internal links — add 3 links to relevant commercial pages
  • ✓ Image alt text — update with keyword-aware descriptions
